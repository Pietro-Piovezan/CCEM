Resumo sobre alelos

O que é um gene:

Uma porção do DNA que guarda informação suficiente para a célula sintetizar proteína.

O que é um alelo

Uma variação do gene no locus gênico

Nomeclatura de alelos: Os alelos são representados por letras minisculas(A, B, etc) ou mainúsculas(a, b, etc), respectivamente representando que o alelo é recessivo ou dominante. O ser humano por ser 2n os genes são compostos por dois alelos

    Tipos de relações de alelos

  • Homozigoto - AA, aa

    Não apenas um dos alelos é dominante

  • Heterozigoto - Aa

    Um dos alelos é dominante

  • Dominância completa

    Um alelo domina completamente o outro

  • Codominânciia

    Ambos os alelos são expressos

  • Dominância incompleta

    Surge uma caracteristica intermediaria

  • Alelos múltipio

    Mais de dois alelos para uma característica
