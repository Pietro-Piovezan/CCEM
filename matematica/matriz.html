Resumo sobre matriz

Comeclatura:

Nome da matriz = começa com uma letra minuscula e é acompanhado por mxn
m = número de linhas, n = número de colunas

Elemento de matriz = representado pela mesma letra, porém minúsculo

coordenada de um elemento da matriz = 'letra minúscula m,n


Matiz transporta

At m e n invertem (m = coluna e n = linha)


Matriz identitaria

Ix
I = identifica que se trata e uma matriz identitaria, x = número de colunas e linhas
Diagonal principal, todos os elementos = 1. Elementos adjacentes = 0



Operações com matriz

  • Multiplicação por número real = todos os elementos são multiplicados
  • Soma de duas matrizes = todos os elementos são somados pelos valores em coordenadas respectivas * a soma só pode ser feita por matrizes com m e n iguais em ambas as operandas
  • Multiplicação de matrizes = soma dos produtos entre os elementos equivalente
    Equivalentes nesse caso se refere a por exemplo, a1,1 = b1,1; a1,2 = b2,1 e a1,3 = b3,1. Essa operação sera o valor c1,1 * só é possivel se o n da primeira for igual ao m da segunda
  • Determinante de matrizes (det()) produto das diagonais decendentes menos o produto das diagonal cresentes. Se a matriz for de mais que segunda ordem, adicione linhas até o número de diagonais completas seja igual a n.
