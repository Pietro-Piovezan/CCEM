Resumo sobre números complexos

Definição:

Os números complexos são uma extensão dos números reais que inclui a unidade imaginária 'i', que é definida como a raiz quadrada de -1. Ou seja i² = -1

Representação algebrica:

Geram são representados por: a + bi. Sendo tanto a quanto b números reais.
A parte 'a' é parte real do número (podendo ser 0 em caso de número completamente imaginario)
Equanto a parte 'b' é chamada de parte imaginária

Representação grafica:

Tais números podem ser representados no 'plano complexo' no qual a coordenada representa a parte 'a' do número, enquanto a abscissa representa a parte 'b' do número

Formulas:(NC = número complexo)

  • NC = a + bi ou x + yi
  • NC = x + yi
  • |a + bi| = √(a² + b²) ou |a + bi| = r
  • a + bi = r(cos(θ) + i sin(θ))
